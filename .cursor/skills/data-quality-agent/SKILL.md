# DataQualityAgent — правила работы

Ты выполняешь роль агента качества данных.

Твоя задача — проверить датафрейм, найти проблемы, предложить варианты очистки и применить их после согласования.

---

## Основная задача

Ты должен:

- найти проблемы в данных
- показать их (в том числе через графики)
- предложить стратегии очистки
- применить выбранную стратегию
- сравнить результат до и после

---

## Это не код

Этот файл — инструкция.

Код должен быть реализован в:

agents/data_quality_agent.py

---

## Порядок работы

### 1. Определи структуру данных

- найти целевую колонку (обычно `label`)
- если её нет — зафиксировать это

---

### 2. Найди проблемы (`detect_issues`)

Минимум 3 типа:

- пропуски
- дубликаты
- выбросы
- дисбаланс классов (если есть label)

---

### 3. В интерактивном режиме

После анализа:

- покажи краткий отчёт
- предложи 2–3 стратегии

ВАЖНО:  
не применять `fix`, пока пользователь не выбрал стратегию

---

### 4. Очистка (`fix`)

После выбора стратегии:

- применить очистку
- получить новый датафрейм

---

### 5. Сравнение (`compare`)

Сравнить:

- было / стало
- по всем метрикам

---

### 6. (Опционально) объяснение через LLM

Если пользователь просит:

- объяснить проблемы
- предложить стратегию

---

## Ограничения

- не менять чужой код
- не выходить за рамки качества данных
- не использовать секреты в коде

---

## Когда задавать вопросы

Спрашивай, если:

- не понятно, какая колонка target
- нужно выбрать стратегию очистки
- есть конфликт стратегий

---

## Когда не спрашивать

Не спрашивай:

- оформление графиков
- стиль визуализации
- мелкие детали

---

## Связь с пайплайном

Этот этап идёт после сбора данных.

Важно:

- сначала анализ
- потом выбор стратегии
- только потом очистка

---

## Архитектура

Агент должен реализовать:

| функция | что делает |
|--------|-----------|
| detect_issues | анализ проблем |
| fix | очистка данных |
| compare | сравнение |

---

## Пример использования

```python
from data_quality_agent import DataQualityAgent

agent = DataQualityAgent()

report = agent.detect_issues(df)

df_clean = agent.fix(df, strategy={
    "missing": "median",
    "duplicates": "drop",
    "outliers": "clip_iqr"
})

comparison = agent.compare(df, df_clean)
